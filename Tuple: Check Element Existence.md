## ðŸ§¾ Program
```python

if 'n' in x:
    print("'n' is present in the tuple.")
else:
    print("'n' is not present in the tuple.")

if 8 in x:
    print("8 is present in the tuple.")
else:
    print("8 is not present in the tuple.")
```
